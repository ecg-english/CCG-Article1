Level 0: THE CORE

1. 発音・声調（四声）の変調ルール

ただの発音ではなく、「不」や「一」の声調変化（変調）まで網羅する

2. 基本語順 (SVO) と「場所」

肯定文・否定文。日本語と違う「場所は動詞の前（在〜）」の徹底

3. 判断・存在・所有 (是 / 在 / 有)

この3大動詞の使い分けを完璧にする（「ある」は「在」か「有」か）

4. 人称代名詞と「的」 (所有)

私、私たち、私の〜

5. 指示代名詞とこそあど (这 / 那 / 哪)

これ、あれ、どれ、ここ、そこ、どこ

6. 単純疑問文 (吗 / 呢)

「〜ですか？」「じゃあ〜は？(省略疑問)」

7. 疑問詞疑問文 (5W1H)

什么(何), 谁(誰), 什么时候(いつ), 哪里(どこ), 为什么(なぜ), 怎么(どうやって)

8. 数詞と量詞の基礎

数字の数え方。「二」と「两」の違い。基本の「个」
